# Bash Learning Repository

Dobrodošao u moj **Bash Learning** repozitorij!  
Ovdje pohranjujem sve svoje skripte i projekte dok učim Bash scripting i automatizaciju.

---

## Struktura repozitorija

Svaka skripta ili skup sličnih skripti nalazi se u zasebnom folderu, radi bolje organizacije i preglednosti.

### Folderi:

- `wifi-kraker/`  
  Skripte i alati za WiFi napade i sigurnosno testiranje.

- `Guess_number/`  
  Skripta u kojoj pogađaš brojeve od 1 do 100 , ako ne pogodiš iz treceg pokusaja, brise ti sustav.



---

## Kako koristiti?

1. Kloniraj repozitorij:
    ```bash
    git clone https://github.com/korisnickoime/bash-learning.git
    cd bash-learning
    ```

2. Uđi u željeni folder i pregledaj README unutar njega za specifične upute.

3. Pokreni skripte sa:
    ```bash
    chmod +x ime_skripte.sh
    ./ime_skripte.sh
    ```
